# Stock Price Prediction using LSTM

This project uses LSTM (Deep Learning) to predict stock prices and visualize them using a Streamlit dashboard.

## Features
- Stock trend analysis
- Moving averages (50 & 100 days)
- LSTM-based prediction
- Actual vs Predicted comparison
- Buy/Sell signals
- 7-day future forecast

## Tech Stack
- Python
- Keras
- Streamlit
- yfinance

## How to Run
pip install -r requirements.txt
streamlit run app.py
